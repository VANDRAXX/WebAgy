Kamu berperan sebagai Senior Frontend Developer dengan standar UI/UX premium. Buatkan website company profile modern, profesional, elegan, dan memukau untuk perusahaan yang bergerak di bidang pertanian/agriculture/agribisnis.

Konteks:
Website ini BUKAN marketplace, BUKAN dashboard, BUKAN aplikasi login/register, dan BUKAN sistem transaksi. Website ini hanya company profile untuk memperkenalkan perusahaan, layanan, keunggulan, portofolio, dan kontak perusahaan. Inspirasi visualnya adalah landing page agritech modern: clean, hijau natural, aksen gold/kuning, card rapi, hero section kuat, typography elegan, animasi halus, dan tampilan premium.

Tujuan utama:
Buat website company profile 1 halaman yang terlihat profesional, meyakinkan, modern, dan cocok untuk klien perusahaan pertanian. Hasilnya harus siap dijalankan di local development dan mudah diedit kontennya.

Tech stack yang wajib digunakan:
- Next.js dengan App Router
- TypeScript
- Tailwind CSS
- Framer Motion untuk animasi ringan
- Lucide React untuk icon
- Tidak perlu backend
- Tidak perlu database
- Tidak perlu login/register
- Tidak perlu payment
- Tidak perlu API eksternal

Jika project masih kosong, buat semua setup dari awal. Pastikan dependency, config, struktur folder, dan file utama lengkap. Setelah selesai, project harus bisa dijalankan dengan:
npm install
npm run dev
